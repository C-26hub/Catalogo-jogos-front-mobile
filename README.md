# 🎮 Meus Jogos

Aplicativo mobile desenvolvido em React Native com Expo para gerenciamento de uma biblioteca pessoal de jogos.

O aplicativo permite cadastrar, listar, editar e excluir jogos consumindo uma API REST desenvolvida em Node.js, Express e MongoDB.

## Funcionalidades

* Cadastrar jogos
* Listar jogos
* Editar jogos
* Excluir jogos
* Definir status do jogo

  * Quero Jogar
  * Jogando
  * Finalizado
* Atribuir nota de 0 a 10
* Interface com tema gamer e modo escuro

## Tecnologias Utilizadas

* React Native
* Expo
* Axios
* JavaScript

## Pré-requisitos

* Node.js
* npm
* Expo Go (Android ou iOS)

## Instalação

Clone o repositório:

```bash
git clone <URL_DO_REPOSITORIO>
```

Entre na pasta do projeto:

```bash
cd meus-jogos
```

Instale as dependências:

```bash
npm install
```

## Execução

Inicie o projeto:

```bash
npx expo start
```

Após iniciar, escaneie o QR Code utilizando o aplicativo Expo Go.
